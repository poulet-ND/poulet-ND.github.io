Comment installer linux à côté de windows

Ici nous allons voir comment installer linux à côté de windows. Avant de commencer, n'oublier pas de faire une copie de vos dossiers sur un disque dur externe ou sur un compte cloud temporaire. Installer linux n'engendre normalement aucun risque mais c'est une précaution à prendre si jamais vous faites une mauvaise manoeuvre.

Sommaire:

  • Matèriel nécessaire
  1. Télécharger la distribution linux
  2. Booter sa clef USB
  3. Lancer et installer Linux
  • Conclusion

Matèriel nécessaire

  • Une clef Usb d'au moins 4 Go (Suffisante pour y mettre le fichier Linux)
  • L'ordinateur pour y installer Linux

Télécharger la distribution Linux

Choisissez votre distribution. La plus connu est Ubuntu mais pour la simplicité de l'interface, je vous conseille d'opter plutôt pour LinuxMint.

Dans la meme distribution Linux, il y a en général plusieurs environnement ou "flavors". Ce sont les différents moteurs d'interface. Choisissez celle que vous voulez. Sur LinuxMint, je vous conseille Cinnamon. N'oubliez pas de regarder les exigences systèmes (par exemple celles de Mint Cinnamon, Mint XFCE, Ubuntu, Bodhi Linux), si l'ordinateur est un peu vieux.

Mint Xfce est engénéralement plus léger mais si vous êtes vraiment très limité en mémoire et en processeur, vous pouvez essayer Bodhi Linux qui est-une distribution très compacte.

Téléchargez le fichier ISO

Booter sa clef USB

  1. Installez un logiciels pour booter les clef USB (exemple: Rufus)
  2. Connecter sa clef USB
  3. Lancer le logiciel
  4. Sélectionnez le fichier ISO comme sélection boot et la clef USB comme destination
  5. Mettez "BIOS UEFI" comme cible/target.
  6. Laissez les autres paramètres tels quels et éxécutez le logiciels.
  7. Si vous recevez des messages d'erreurs qui expliquent que le boot de la clef ne sera complète, continuez. Si la machine propose une solution, acceptez.

Lancer Linux

Connecter la clef USB à l'ordinateur, puis booter l'ordinateur depuis la clef USB. Sur windows 10, on peut faire cela en se connectant puis appuyant sur la touche majeur et sur le bouton redémarrer (en bas à gauche). Sinon vous pouvez le faire en modifiant les paramètre BIOS (le processus diffère en fonction du système d'exploitation) puis en faisant redémarrer l'ordinateur.

Une fois arrivé jusque là, vous avez normalement un menu pour soit installer Linux, soit l'essayer. Choisissez d'essayer, vous ne risquez rien. Si jamais la distribution ne vous plait pas, vous pourrez redémarrer avec le bouton de la distribution et surtout pas le bouton qui se trouve sur l'ordinateur

Installer Linux

Normalement, dans votre nouvel environnement, vous avez un logiciel "Installer Linux" sur votre fond d'écran, Cliquez dessus.

Une fois avoir démarré le logiciel:

  1. On vous demandera le clavier de votre ordinateur (US, FR, NL, AU, etc.) et le mode d'écriture associé
  2. Vous choisirez aussi la zone horaire, la langue du système, le mot de passe, etc.
  3. Quand vous aurez l'option d'installation, choisissez l'installation dans une partition à côté de windows. Faites attention car si vous choisissez d'installer à la place de windows, cela vous éffacera tous les fichier que vous aviez stoqué sur windows.
  4. Continuez à répondre au questionnaire jusqu'à ce que le logiciel passe à l'action.

Là, vous passienterez un petit moment avant de pouvoir redémarrer avec Linux ou Windows.

Conclusion

Installer Linux à côté de windows ne demande pas beaucoup de matèriel à part une clef USB, par contre, cela demande un peu de temps (une petite heure). Lorsque vous installez Linux vous avez beaucoup de choix. Cependant l'installation de LinuxMint et la plus conviviale. Si vous avez un ordinateur un peu ancien, regardez des distributions et environnement plus légèrs Ainsi que leurs exigence système.
